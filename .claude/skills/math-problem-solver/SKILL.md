# 数学问题解答 Skill

当用户提出数学相关问题时，按照以下规范进行回答。

## ⚠️ 核心规则（必须遵守）

1. **绝对禁止**将公式放入 ` ``` ` 或 ` ```latex ` 代码块中
2. **绝对禁止**使用反引号 `` ` `` 包裹任何公式
3. **必须**直接使用 Unicode 数学符号书写公式，让用户无需任何渲染即可直接阅读
4. 所有回答使用**中文**
5. 解题过程用**自然语言 + 公式混排**，像手写在纸上一样让人直接看懂

## Unicode 数学符号规范

### 直接书写规则

不要写 LaTeX 代码，直接使用 Unicode 字符：

| ❌ 禁止（LaTeX 代码） | ✅ 正确（Unicode 直写） |
|----------------------|----------------------|
| `$\frac{a}{b}$` | a/b 或 (a + b)/(c + d) |
| `$x^2$` | x² |
| `$x_i$` | xᵢ |
| `$\sqrt{x}$` | √x |
| `$\alpha$` | α |
| `$\int f(x)dx$` | ∫f(x)dx |
| `$\sum_{i=1}^{n}$` | Σᵢ₌₁ⁿ |
| `$\lim_{x \to 0}$` | lim(x→0) |
| `$\infty$` | ∞ |
| `$\leq$` | ≤ |
| `$\neq$` | ≠ |
| `$\in$` | ∈ |
| `$\partial$` | ∂ |

### 上标数字和字母

使用 Unicode 上标：⁰ ¹ ² ³ ⁴ ⁵ ⁶ ⁷ ⁸ ⁹ ⁿ ⁺ ⁻ ⁽ ⁾ ˣ

示例：x² + y² = r²，eⁱᵖ + 1 = 0

### 下标数字和字母

使用 Unicode 下标：₀ ₁ ₂ ₃ ₄ ₅ ₆ ₇ ₈ ₉ ₊ ₋ ₌ ₍ ₎ ₐ ₑ ᵢ ⱼ ₖ ₙ ₓ

示例：a₁ + a₂ + ... + aₙ

Α Β Γ Δ Ε Ζ Η Θ Ι Κ Λ Μ Ν Ξ Π Ρ Σ Τ Υ Φ Χ Ψ Ω

### 运算符和关系符

- 运算：＋ － × ÷ ± ∓ · ∘
- 关系：= ≠ ≈ ≡ < > ≤ ≥ ≪ ≫ ∝
- 集合：∈ ∉ ⊂ ⊃ ⊆ ⊇ ∪ ∩ ∅
- 逻辑：∀ ∃ ¬ ∧ ∨ ⇒ ⇔
- 微积分：∫ ∬ ∮ ∂ ∇ Σ Π ∏
- 箭头：→ ← ↔ ⇒ ⇐ ⇔ ↦

### 分数的写法

- 简单分数：a/b
- 复杂分数：用括号明确层级，如 (a + b) / (c + d)
- 连分数：逐层缩进书写

### 矩阵/行列式的写法

用方括号或竖线 + 整齐排列：

```
    ┌         ┐
A = │ a₁₁  a₁₂ │
    │ a₂₁  a₂₂ │
    └         ┘
```

或简写为：A = [a₁₁, a₁₂; a₂₁, a₂₂]

行列式用竖线：|A| = a₁₁·a₂₂ − a₁₂·a₂₁

### 多行推导的写法

使用对齐的等号链：

  f(x) = (x + 1)²
       = x² + 2x + 1
       = (x + 1)(x + 1)

## 完整示例

---

**📝 题目**

求不定积分 ∫ x/√(1+x²) dx。

**💡 解题思路**

注意到被积函数的分子 x 恰好是分母中 1+x² 求导后的一部分，因此考虑**凑微分法**。

**📐 详细步骤**

**第一步：识别微分关系**

注意到 d(1+x²) = 2x·dx，即 x·dx = (1/2)·d(1+x²)。

**第二步：换元**

令 u = 1 + x²，则：

∫ x/√(1+x²) dx = (1/2) · ∫ du/√u

**第三步：计算积分**

(1/2) · ∫ u⁻¹ᐟ² du = (1/2) · 2u¹ᐟ² + C
                     = √u + C

**第四步：回代**

将 u = 1 + x² 代回，得到：

∫ x/√(1+x²) dx = √(1+x²) + C

**✅ 最终答案**

**∫ x/√(1+x²) dx = √(1+x²) + C**
