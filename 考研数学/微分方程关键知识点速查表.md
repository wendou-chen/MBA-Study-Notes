## 📐 积分因子法

### 一阶线性微分方程标准形式
$$y' + P(x)y = Q(x)$$

**积分因子**：$\mu(x) = e^{\int P(x)dx}$

**通解公式**：
$$y = e^{-\int P(x)dx}\left[C + \int Q(x)e^{\int P(x)dx}dx\right]$$

### 常见形式速查

| 方程形式 | 积分因子 | 关键变换 |
|---------|---------|---------|
| $y' + y = f(x)$ | $e^x$ | $(ye^x)' = e^x f(x)$ |
| $y' - y = f(x)$ | $e^{-x}$ | $(ye^{-x})' = e^{-x} f(x)$ |
| $y' + \frac{1}{x}y = f(x)$ | $x$ | $(xy)' = xf(x)$ |
| $y' - \frac{1}{x}y = f(x)$ | $\frac{1}{x}$ | $(\frac{y}{x})' = \frac{f(x)}{x}$ |

## 🔄 变限积分求导（Leibniz公式）

### 基本形式
$$\frac{d}{dx}\int_{a(x)}^{b(x)} f(t)dt = f(b(x))b'(x) - f(a(x))a'(x)$$

### ⚡ 变限积分等式求导：固定套路（两个选项）

**遇到变限积分等式，立即判断：**

```
变限积分等式
    ↓
判断积分限形式
    ↓
┌─────────────────┬─────────────────┐
│  选项1：单侧含x  │  选项2：双侧含x  │
│  (如 ∫₀ˣ, ∫ₐˣ)  │  (如 ∫ₓˣ⁺ᵃ)     │
└─────────────────┴─────────────────┘
         ↓                   ↓
    直接对x求导         对参数求导
         ↓                   ↓
    f(x) = ...         令参数=特殊值
                            ↓
                       f'(x) = ...
```

### 选项1：单侧含 $x$ → 直接对 $x$ 求导

**适用**：$\int_0^x$、$\int_a^x$、$\int_x^b$ 等（只有一个限含 $x$）

**例子**：
$$g(x) = \int_0^x [1-f(t)]dt + 1$$

直接对 $x$ 求导：
$$g'(x) = 1 - f(x)$$

### 选项2：双侧含 $x$ → 对参数求导 + 令参数为特殊值

**适用**：$\int_x^{x+a}$、$\int_{x-a}^{x+a}$ 等（上下限都含 $x$）

**步骤**：
1. 对参数求导（如对 $a$ 求导）
2. 令参数为特殊值（如 $a=0$）消去积分

**例子**：
$$f(x+a) = \int_x^{x+a} \frac{t(t^2+1)}{f(t)}dt + f(x)$$

- ❌ 直接对 $x$ 求导：$f'(x+a) = f(x+a) - f(x) + f'(x)$（复杂）
- ✅ 对 $a$ 求导：$f'(x+a) = f(x+a)$（简单）
- ✅ 令 $a=0$：$f'(x) = \frac{x(x^2+1)}{f(x)}$（积分消失）

### 常见应用

**形式1**：$\frac{d}{dx}\int_0^x f(t)dt = f(x)$

**形式2**：$\frac{d}{dx}\int_x^{x+a} f(t)dt = f(x+a) - f(x)$

**形式3**：$\frac{d}{da}\int_x^{x+a} f(t)dt = f(x+a)$

### 💡 关键技巧：上下限都含 $x$ 时对参数求导

**问题**：当积分上下限都含 $x$ 时（如 $\int_x^{x+a} f(t)dt$），直接对 $x$ 求导很复杂

**解决方法**：
1. 引入参数（如 $a$）
2. 对参数求导（只需处理上限）
3. 令参数为特殊值（如 $a=0$）消去积分

**例子**：
$$f(x+a) = \int_x^{x+a} \frac{t(t^2+1)}{f(t)}dt + f(x)$$

- ❌ 直接对 $x$ 求导：需要同时处理 $\int_x^{x+a}$ 的上下限，非常复杂
- ✅ 对 $a$ 求导：$\frac{\partial}{\partial a}f(x+a) = f(x+a) \cdot 1$（只处理上限）
- ✅ 令 $a=0$：得到 $f'(x) = \frac{x(x^2+1)}{f(x)}$（积分消失）

**记忆口诀**：
- 单侧含 $x$ → 直接对 $x$ 导
- 双侧含 $x$ → 对参数导 + 赋特值

## 📊 曲率公式

### 平面曲线曲率
$$\kappa = \frac{|y''|}{(1+y'^2)^{3/2}}$$

### 特殊情况
- 当 $y' = 0$ 时：$\kappa = |y''|$
- 圆的曲率：$\kappa = \frac{1}{R}$（$R$ 为半径）

## 🔺 变量代换公式

### $x$ 与 $y$ 互换
设 $x = x(y)$，则：

$$\frac{dy}{dx} = \frac{1}{x'_y}$$

$$\frac{d^2y}{dx^2} = -\frac{x''_y}{(x'_y)^3}$$

### 齐次方程换元
对于 $y' = f(\frac{y}{x})$，令 $u = \frac{y}{x}$：

$$y = ux \Rightarrow y' = u + xu'$$

代入得：$u + xu' = f(u)$，即 $\frac{du}{f(u)-u} = \frac{dx}{x}$

---

## 🎯 二阶微分方程降阶

### 缺 $y$ 型：$F(x, y', y'') = 0$
令 $p = y'$，则 $y'' = p'$，降为一阶方程 $F(x, p, p') = 0$

### 缺 $x$ 型：$F(y, y', y'') = 0$
令 $p = y'$，则 $y'' = p\frac{dp}{dy}$，降为一阶方程 $F(y, p, p\frac{dp}{dy}) = 0$

---

## 🧮 常用积分公式

### 指数三角混合积分
$$\int e^{ax}\cos bx\,dx = \frac{e^{ax}}{a^2+b^2}(a\cos bx + b\sin bx) + C$$

$$\int e^{ax}\sin bx\,dx = \frac{e^{ax}}{a^2+b^2}(a\sin bx - b\cos bx) + C$$

**特例**（$a=-1, b=1$）：
$$\int e^{-x}\cos x\,dx = -\frac{1}{2}e^{-x}(\cos x + \sin x) + C$$

$$\int e^{-x}\sin x\,dx = -\frac{1}{2}e^{-x}(\sin x - \cos x) + C$$

### 反三角函数积分
$$\int \frac{dx}{1+x^2} = \arctan x + C$$

$$\int \frac{dx}{\sqrt{1-x^2}} = \arcsin x + C$$

## 📈 欧拉方程

### 标准形式
$$x^2y'' + pxy' + qy = f(x)$$

### 求解方法
**换元**：令 $x = e^t$（或 $t = \ln x$），则：

$$x\frac{dy}{dx} = \frac{dy}{dt}$$

$$x^2\frac{d^2y}{dx^2} = \frac{d^2y}{dt^2} - \frac{dy}{dt}$$

代入后转化为常系数线性微分方程。

## 🎲 特解形式判断（共振情况）

### 非齐次项 $f(x) = P_m(x)e^{\lambda x}$

| 特征根情况 | 特解形式 |
|-----------|---------|
| $\lambda$ 不是特征根 | $y^* = Q_m(x)e^{\lambda x}$ |
| $\lambda$ 是单根 | $y^* = xQ_m(x)e^{\lambda x}$ |
| $\lambda$ 是重根 | $y^* = x^2Q_m(x)e^{\lambda x}$ |

其中 $Q_m(x)$ 是 $m$ 次多项式。

### 等比数列

## 🌐 全微分条件

### 判断条件
$du = P(x,y)dx + Q(x,y)dy$ 是全微分 $\Leftrightarrow$ $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$

### 求原函数方法
**偏积分法**：
$$u(x,y) = \int P(x,y)dx + \varphi(y)$$

对 $y$ 求偏导，令其等于 $Q(x,y)$，确定 $\varphi(y)$。

## 📐 拉普拉斯算子

### 直角坐标
$$\nabla^2 u = \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2}$$

### 极坐标
$$\nabla^2 u = \frac{\partial^2 u}{\partial r^2} + \frac{1}{r}\frac{\partial u}{\partial r} + \frac{1}{r^2}\frac{\partial^2 u}{\partial \theta^2}$$

## 🎯 几何应用常用公式

### 切线方程
在点 $(x_0, y_0)$ 处：$Y - y_0 = y'(x_0)(X - x_0)$

### 法线方程
在点 $(x_0, y_0)$ 处：$Y - y_0 = -\frac{1}{y'(x_0)}(X - x_0)$

### 弧长微分
$$ds = \sqrt{1+(y')^2}dx$$

### 旋转体体积
绕 $x$ 轴：$V = \pi\int_a^b y^2dx$

绕 $y$ 轴：$V = 2\pi\int_a^b xy\,dx$

## 💡 常见错误提醒

### ❌ 易错点1：积分因子符号
- $y' + P(x)y = Q(x)$ 的积分因子是 $e^{\int P(x)dx}$（**正号**）
- $y' - P(x)y = Q(x)$ 的积分因子是 $e^{-\int P(x)dx}$（**负号**）

### ❌ 易错点2：变限积分求导
- 上限函数求导：$\frac{d}{dx}\int_a^{g(x)} f(t)dt = f(g(x)) \cdot g'(x)$
- **不要忘记乘以 $g'(x)$！**

### ❌ 易错点3：曲率公式分母
- 曲率公式分母是 $(1+y'^2)^{3/2}$，**不是** $(1+y'^2)^{1/2}$

### ❌ 易错点4：欧拉方程换元
- 换元 $x = e^t$ 后，$x^2y'' = y''_t - y'_t$
- **不是** $y''_t$，要减去 $y'_t$

## 🔑 解题技巧总结

### 技巧1：识别积分因子
看到 $y' \pm \frac{k}{x}y$ 形式，立即想到积分因子 $x^{\pm k}$

### 技巧2：对参数求导消积分 ⭐⭐⭐
**适用场景**：积分上下限都含 $x$（如 $\int_x^{x+a}$、$\int_{x-a}^{x+a}$）

**操作步骤**：
1. 引入参数（通常题目已给出）
2. 对参数求导（避免同时处理上下限）
3. 令参数为特殊值（如 $a=0$）消去积分

**为什么有效**：
- 对 $x$ 求导：$\frac{d}{dx}\int_x^{x+a} = f(x+a) - f(x)$（需处理两个端点）
- 对 $a$ 求导：$\frac{d}{da}\int_x^{x+a} = f(x+a)$（只需处理上限）

### 技巧3：几何条件代数化
- 切线、法线 → 写出方程
- 距离、面积 → 建立等式
- 曲率 → 用公式表达

### 技巧4：降阶优先
二阶方程缺 $x$ 或缺 $y$ 时，优先考虑降阶

### 技巧5：特解形式记忆
非齐次项含 $e^{\lambda x}$，检查 $\lambda$ 是否为特征根，决定是否乘 $x$
