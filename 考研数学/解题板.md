**📝 题目**

设 $f(x)$ 可导，对任何实数 $x,y$ 满足 $f(x+y) = e^x f(y) + e^y f(x)$，且 $f'(0) = e$，求 $f(x)$。

**💡 解题思路**

处理含有 $f(x+y)$ 的抽象函数方程求解析式问题。通过**偏导数法**将等式转化为一阶线性常微分方程，结合初始条件求解。

**📐 详细步骤**

**第一步：找初始条件**

令 $x = 0, y = 0$ 代入原方程：

$$f(0+0) = e^0 f(0) + e^0 f(0)$$

$$f(0) = 2f(0) \implies f(0) = 0$$

**第二步：偏导法构造微分方程**

将 $x$ 当作常数，等式两边同时对 $y$ 求偏导：

$$\frac{\partial}{\partial y} [f(x+y)] = \frac{\partial}{\partial y} [e^x f(y) + e^y f(x)]$$

$$f'(x+y) = e^x f'(y) + e^y f(x)$$

令 $y=0$，代入已知条件 $f'(0) = e$：

$$f'(x) = e^x \cdot e + 1 \cdot f(x)$$

移项整理，得到一阶线性常微分方程：

$$f'(x) - f(x) = e^{x+1}$$

**第三步：解一阶常微分方程**

同乘积分因子 $e^{\int -1\, dx} = e^{-x}$：

$$(e^{-x}f(x))' = e$$

两边对 $x$ 积分：

$$e^{-x}f(x) = ex + C$$

利用 $f(0) = 0$，令 $x=0$ 得 $C = 0$，两边同乘 $e^x$：

$$f(x) = x e^{x+1}$$

**✅ 最终答案**

$$\boxed{f(x) = x e^{x+1}}$$

## 🤔 疑难辨析（2026-02-20）

**Q：这道题怎么想到用偏导法的？看到什么条件就该想到什么做法？**
> 用户希望建立「条件 → 做法」的自动化链路

**A：条件链路分析**

核心链路只有一条：**看到 $f(x+y)$ → 偏导降维**

- 看到 **$f(x+y)$ 型函数方程**（等式中同时出现 $x, y$ 两个自变量）→ 立刻想到：对其中一个变量求导，再令另一个变量 $=0$，将两元方程「坍缩」为一元 ODE
- 看到 **$f'(0) = e$** → 这是代入用的初始条件，确认偏导路线可行（导完令 $y=0$ 时正好用上）
- 得到 **$f'(x) - f(x) = e^{x+1}$** → 一阶线性 ODE，积分因子法，标准套路

一句话记忆：**凡是 $f(x+y)$ 型，核心操作就是「偏导降维」—— 对一个变量求导，令另一个为零，两元变一元。**

**Q：第三步的“积分因子”跟上图的一阶线性方程公式有什么区别？为什么叫法看起来不一样？**

**A：本质没有区别：上图是通式；题解第三步是在通式里把 $p(x),q(x)$ 换成了本题的具体函数。**

- 上图对应的标准形是

$$y' + p(x)y = q(x)$$

- 本题第三步得到的是

$$f'(x) - f(x) = e^{x+1}\;\;\Longleftrightarrow\;\; f'(x) + (-1)\,f(x)=e^{x+1}$$

所以本题里

$$p(x)=-1,\qquad q(x)=e^{x+1}$$

积分因子就是

$$\mu(x)=e^{\int p(x)\,dx}=e^{\int -1\,dx}=e^{-x}$$

- “积分因子法”的核心目标只有一个：把左边凑成乘积求导

$$(\mu y)'=\mu y' + \mu' y$$

只要让 $\mu'=\mu p(x)$，就能把 $\mu y' + \mu p(x)y$ 变成 $(\mu y)'$。
- 你题解里写的 $e^{-x}$，就是把上图的 $\mu=e^{\int p}$ 算出来；有些书把过程叫“积分因子法”，把结论直接背成“通解公式”，所以你会感觉“名字不一样”。

### 积分因子法（详细版模板）

**触发条件（先判断能不能用）**  
能整理成下面的形式，就能用积分因子法：

$$y' + p(x)y = q(x)$$

等价地，如果是

$$a(x)y' + b(x)y = c(x),\ a(x)\neq 0$$

先两边除以 $a(x)$ 化到标准形即可。

**步骤 1：整理到标准形**  
把所有含 $y$ 的项放左边，右边只留 $x$ 的函数，得到 $y' + p(x)y = q(x)$。

**步骤 2：构造积分因子（这是推导的关键）**  
希望两边同乘一个函数 $\mu(x)$ 后，左边能写成 $(\mu y)'$：

$$(\mu y)'=\mu y' + \mu' y \stackrel{!}{=} \mu y' + \mu p(x) y$$

因此必须满足

$$\mu'=\mu p(x)\;\;\Longleftrightarrow\;\;\frac{\mu'}{\mu}=p(x)$$

两边积分：

$$\ln \mu=\int p(x)\,dx\;\;\Longrightarrow\;\;\mu(x)=e^{\int p(x)\,dx}$$

说明：$\mu$ 乘任何常数因子都可以，最终会被积分常数 $C$ 吸收，不影响通解。

**步骤 3：两边同乘 $\mu$ 并凑成导数**  

$$\mu y' + \mu p(x)y = \mu q(x)\;\;\Longrightarrow\;\;(\mu y)'=\mu q(x)$$

**步骤 4：两边积分**  

$$\mu(x)\,y(x)=\int \mu(x)\,q(x)\,dx + C$$

**步骤 5：解出 $y$ 并代初值（若有）**  

$$y(x)=\frac{1}{\mu(x)}\left[\int \mu(x)\,q(x)\,dx + C\right]$$

### 把模板套回本题（对应第三步）

本题的标准形已经是

$$f'(x) - f(x) = e^{x+1} \quad\Rightarrow\quad p(x)=-1,\; q(x)=e^{x+1}$$

积分因子：

$$\mu(x)=e^{\int -1\,dx}=e^{-x}$$

同乘并凑导数：

$$(e^{-x}f(x))' = e^{-x}\cdot e^{x+1} = e$$

积分并代 $f(0)=0$：

$$e^{-x}f(x)=ex+C,\quad C=0\Rightarrow f(x)=xe^{x+1}$$
