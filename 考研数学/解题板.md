**💡 解题思路**

这道题的题眼是**几何条件翻译成 ODE**：「到原点距离 = 切线在 $y$ 轴截距」直接写成 $\sqrt{x^2+y^2} = y - xy'$，化简后是齐次方程，令 $v = y/x$ 分离变量即可。第 (II) 问是经典的「切线与坐标轴围成三角形面积最小」优化问题。
📋 固定模式：[[固定解法速查表#几何体眼翻译（切线/拐点/极值）]] + [[固定解法速查表#齐次方程换元法]] + [[固定解法速查表#可分离变量方程]]

**📐 详细步骤**

---

### (I) 求曲线 $L$ 的方程

**第一步：翻译几何条件为微分方程**

曲线上任意一点 $P(x, y)$ 处的切线方程为：
$$Y - y = y'(X - x)$$
令 $X = 0$，得切线在 $y$ 轴上的截距为：
$$Y\big|_{X=0} = y - xy'$$

点 $P$ 到原点的距离为 $\sqrt{x^2 + y^2}$。

由题意「距离 = 截距」，得：
$$\sqrt{x^2 + y^2} = y - xy'$$

整理：
$$xy' = y - \sqrt{x^2 + y^2} \implies y' = \frac{y - \sqrt{x^2+y^2}}{x}$$
**第二步：齐次方程换元**

右端可以写成 $y/x$ 的函数形式，这是齐次方程！令 $v = \dfrac{y}{x}$，则 $y = vx$，$y' = v + xv'$。

代入：
$$v + xv' = \frac{vx - x\sqrt{1+v^2}}{x} = v - \sqrt{1+v^2}$$

消去 $v$，得到极其干净的可分离方程：
$$xv' = -\sqrt{1+v^2}$$

**第三步：分离变量并积分**

$$\frac{dv}{\sqrt{1+v^2}} = -\frac{dx}{x}$$

两边积分（左边是标准公式 $\int \frac{dv}{\sqrt{1+v^2}} = \ln(v + \sqrt{1+v^2})$）：

$$\ln\!\left(v + \sqrt{1+v^2}\right) = -\ln x + \ln C_1$$

即：
$$v + \sqrt{1+v^2} = \frac{C_1}{x}$$

回代 $v = y/x$：
$$y + \sqrt{x^2+y^2} = C_1$$

**第四步：代入初始条件**

$L$ 过点 $\left(\dfrac{1}{2},\, 0\right)$：
$$0 + \sqrt{\frac{1}{4}} = C_1 \implies C_1 = \frac{1}{2}$$

所以 $\sqrt{x^2+y^2} = \dfrac{1}{2} - y$（要求 $y \leq \dfrac{1}{2}$），两边平方：

$$x^2 + y^2 = \frac{1}{4} - y + y^2$$

**✅ 第 (I) 问答案**

$$\boxed{y = \frac{1}{4} - x^2 \quad (x > 0)}$$

---
### (II) 求使切线与 $L$ 及两坐标轴所围面积最小的切线

**第一步：建立切线方程**

曲线 $y = \dfrac{1}{4} - x^2$ 在第一象限部分：$0 < x < \dfrac{1}{2}$，$y' = -2x$。

在点 $(x_0,\, y_0)$ 处的切线（其中 $y_0 = \dfrac{1}{4} - x_0^2$）：
$$Y = -2x_0\, X + x_0^2 + \frac{1}{4}$$

**第二步：求截距**

- $y$ 轴截距（令 $X=0$）：$B = \dfrac{4x_0^2 + 1}{4}$
- $x$ 轴截距（令 $Y=0$）：$A = \dfrac{4x_0^2 + 1}{8x_0}$

**第三步：建立面积函数**

由于曲线上凸（$y'' = -2 < 0$），切线在曲线上方。切线与两坐标轴围成三角形，曲线下方面积固定，因此最小化三角形面积等价于最小化总围合面积。

$$S = \frac{1}{2} A \cdot B = \frac{(4x_0^2+1)^2}{64x_0}$$

**第四步：求最值**

取对数求导：
$$\ln S = 2\ln(4x_0^2+1) - \ln 64 - \ln x_0$$

$$\frac{S'}{S} = \frac{16x_0}{4x_0^2+1} - \frac{1}{x_0}$$

令 $S' = 0$：
$$16x_0^2 = 4x_0^2 + 1 \implies x_0^2 = \frac{1}{12} \implies x_0 = \frac{1}{2\sqrt{3}}$$

**第五步：写出切线方程**

$$x_0 = \frac{1}{2\sqrt{3}}, \quad y_0 = \frac{1}{6}, \quad y' = -\frac{1}{\sqrt{3}}$$

$$y = -\frac{\sqrt{3}}{3}\,x + \frac{1}{3}, \quad S_{\min} = \frac{\sqrt{3}}{18}$$

**✅ 第 (II) 问答案**

$$\boxed{y = -\frac{\sqrt{3}}{3}\,x + \frac{1}{3}, \quad S_{\min} = \frac{\sqrt{3}}{18}}$$
