**💡 解题思路**

核心思路：**面积条件 → 积分方程 → 对 $x$ 求导化为 ODE → 解一阶线性微分方程**

把"曲线弧与弦 $OP$ 围成的面积"写成积分方程，再对 $x$ 求导（Leibniz 公式）消掉积分，化为一阶线性微分方程求解。
📋 固定模式：[[固定解法速查表#Leibniz公式求导法]] + [[固定解法速查表#一阶线性ODE积分因子法]]

**📐 详细步骤**

---

### 第一步：建立积分方程

设曲线为 $y = y(x)$（$0 \le x \le 1$），线段 $OP$ 所在直线为 $Y = \dfrac{y(x)}{x} \cdot t$（$0 \le t \le x$）。

曲线弧在弦上方（向上凸），所围面积为：

$$S(x) = \int_0^x \left[ y(t) - \frac{y(x)}{x} \cdot t \right] dt = x^2$$

展开：

$$\int_0^x y(t)\,dt - \frac{x\,y(x)}{2} = x^2$$

### 第二步：两边对 $x$ 求导（Leibniz 公式）

左边求导：$y(x) - \dfrac{y(x) + x\,y'(x)}{2} = 2x$

化简：

$$\frac{y(x) - x\,y'(x)}{2} = 2x \implies y - xy' = 4x$$

### 第三步：化为标准一阶线性 ODE

$$y' - \frac{1}{x}\,y = -4$$

### 第四步：巧妙观察求解

注意到：

$$\left(\frac{y}{x}\right)' = \frac{xy' - y}{x^2} = \frac{-4x}{x^2} = -\frac{4}{x}$$

两边积分：

$$\frac{y}{x} = -4\ln x + C$$

### 第五步：代入初始条件

由 $A(1,1)$ 在曲线上：$1 = -4\ln 1 + C \implies C = 1$

**✅ 最终答案**

$$\boxed{y = x(1 - 4\ln x), \quad 0 < x \le 1, \quad y(0) = 0}$$
