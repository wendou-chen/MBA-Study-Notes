# MBA Study Notes (考研数学笔记)

This repository contains study notes, wrong question collections, and knowledge graphs for the MBA/Postgraduate Entrance Exam, primarily focusing on Mathematics.

## Project Structure

- **考研数学/**: Core notes, including:
  - **错题/**: Collection of wrong questions and analysis (e.g., Calculus, Linear Algebra).
  - **习题集/**: Problem sets and exercises.
- **Excalidraw/**: Visual diagrams and knowledge graphs (.canvas files).
- **.claude/**: AI assistant configuration and custom skills.

## Features

- **Obsidian Vault**: Managed using Obsidian for bi-directional linking and knowledge management.
- **AI Integration**: Configured with Claude Code skills for automated note organization and pattern extraction.
- **MCP Support**: Includes a `start_obsidian.py` script to run a Model Context Protocol server for AI context.

## Setup

1. Open this folder as a vault in [Obsidian](https://obsidian.md/).
2. (Optional) To use AI features, ensure `python` is installed and configure `.env` with necessary API keys.
