# 英语写作素材库 Skill

## 触发词

- `/素材`
- `添加写作素材`
- `写作模板`

## 工作流

1. 接收用户提供的新素材内容和类型（模板、句型、替换词等）。
2. 读取 `考研英语/写作素材库.md`。
3. 根据素材类型定位到对应的 H2 区域。
4. 追加新素材条目并保持原有排版风格。
5. 更新 frontmatter 中的 `date` 为当前日期。

## 文件约定

- 主文件：`考研英语/写作素材库.md`
- 保持 Obsidian 兼容语法（callout、表格、标题层级不破坏）
