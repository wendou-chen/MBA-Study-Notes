# 英语阅读理解追踪 Skill

## 触发词

- `/阅读`
- `记录阅读`

## 工作流

1. 检查当前日期是否 `>= 2026-07-01`（Phase 3）。
2. 若未到 Phase 3，提示：`阅读训练在强化阶段启用`。
3. 若已到 Phase 3，接收用户输入的阅读记录：题号、题型、正确/错误、错因、用时。
4. 写入 `考研英语/阅读记录/YYYY-MM-DD-N.md`。
5. 更新 `考研英语/阅读追踪.md` 累计统计表。

## 题型分类

- 细节题
- 推断题
- 主旨题
- 态度题
- 词义题

## 累计追踪表格式

```markdown
| 题型 | 总题数 | 正确数 | 正确率 | 趋势 |
|:----:|:------:|:------:|:------:|:----:|
```
