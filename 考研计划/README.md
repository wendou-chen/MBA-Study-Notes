# 考研计划目录说明

> [!info]
> `Kaoyan Countdown` 默认读取本目录下的计划文件。文件名只要以日期开头即可，例如 `2026-03-06.md` 或 `2026-03-06 周五.md`。

## 目录内容

- `示例计划.md`：public 仓自带的演示文件，fresh clone 后可直接被插件读取。
- `Templates/考研每日计划模板.md`：用于生成你自己的每日计划。

## 推荐用法

1. 先打开 `示例计划.md` 看格式。
2. 复制一份为今天日期，例如 `2026-03-06 周五.md`。
3. 按 `- [ ] 时间 | 科目 | 描述` 的三段式继续填写。

## 兼容规则

以下两种任务行都能被 `Kaoyan Countdown` 解析：

- `- [ ] 08:00-10:00 | 数学 | 高数例题 3 题`
- `| 08:00-10:00 | 数学 | 高数例题 3 题 | ⬜ |`
